# Sustainability Reports Website

This project is the final work for my Bachelor's degree in Computer Science (L-31) at **Pegaso Telematic University**. The goal of the project was to create a web platform that allows users to download sustainability reports of a company in the secondary sector.


## Project Overview
The website provides a clean and user-friendly interface where users can easily access and download corporate sustainability reports. It was designed to showcase practical skills in front-end web development and UI/UX design.


## Technologies Used
- **HTML & CSS** – For structure and styling of the web pages.  
- **JavaScript** – Used a small script with **Intersection Observer** to implement smooth scroll animations and interactive elements.


## Features
- Responsive design compatible with multiple devices and screen sizes.  
- Animated scrolling effects to enhance user experience.  
- Easy download access for corporate sustainability reports.


## Learning Outcomes
Through this project, I strengthened my knowledge of:
- Front-end development fundamentals.  
- Basic JavaScript animations and event handling.  
- Creating professional, accessible, and user-friendly web interfaces.
